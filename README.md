# tpp
trading platform python practice  

### 啟動前設定
於./config資料夾內編輯binance.config  
在裡面填上api key及secret  

### 啟動方式  
main class 為 trading_platform.py  
執行後  
輸入start strategy name可開始    
輸入stop strategy name可停止  
輸入exit離開程式   

log位於root  
名稱為log.txt
